## Tech Stack

React, styled-components, recoil



## Installation

Install Frontend with npm

```bash
  npm install
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/Kusitms-28th-Hicardi-D/Frontend.git
```

Go to the project directory

```bash
  cd Frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
